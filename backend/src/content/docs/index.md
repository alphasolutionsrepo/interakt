# Interakt user documentation

End-user documentation for the Interakt admin dashboard. These docs are also indexed into the in-app **Help Assistant** (the ? icon in any admin screen, **Ask** tab).

If you're reading this on GitHub:

- **intro.md** — start here for the platform overview.
- **getting-started/** — *What is Interakt*, the *Initial Setup* walkthrough, and *Your first experience* (an end-to-end with your own data).
- **admin-tour/** — navigating the dashboard, what's on the home page.
- **concepts/** — one page per admin screen / area. Covers every setting and what each one does.
- **guides/** — task walkthroughs: creating an index, configuring synonyms, embedding the widget, configuring guardrails, editing prompts, etc.

The audience is the **business user** of Interakt — the person setting up search and chat experiences for their company, not a developer hosting the platform. For installation, deployment, and contributor docs, see the repository root README.
