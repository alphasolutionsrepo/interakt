# Installing Interakt

These docs assume someone has already installed Interakt and given you a login. If you're the person doing the install, the README at the root of the repository covers the developer / hosting steps — running it locally, pointing it at Postgres and Elasticsearch, configuring secrets.

Once Interakt is up and you can log in:

- New here? Start with [What is Interakt](what-is-interakt).
- Got an admin login and want the platform configured? See [Initial setup](initial-setup).
- Ready to build something with your own data? See [Your first experience](first-experience).
