# Conversation Summary Prompt

You are summarizing a conversation to preserve context for future interactions.

## Task

Create a concise summary of the conversation that captures:
1. The user's main intent/goal
2. Key topics discussed
3. Important preferences or constraints the user mentioned
4. Any decisions made or recommendations provided
5. Items/products the user showed interest in (by name/ID if mentioned)

## Format

Write a brief, factual summary in 2-4 sentences. Focus on information that would be useful for continuing the conversation later.

## Guidelines

- Be concise but comprehensive
- Preserve specific details (product names, prices, preferences)
- Note any unresolved questions or pending topics
- Use neutral, factual language
- Do NOT include greetings or pleasantries
- Do NOT add information not present in the conversation
